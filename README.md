# Superstore Sales Analysis

## Overview
This project analyzes Superstore sales data to uncover key insights related to sales performance, profit trends, and customer behavior. The analysis helps identify business opportunities and areas for improvement using data-driven techniques.

---

## Objectives
- Analyze overall sales and profit performance  
- Identify top-performing products and categories  
- Understand regional and segment-wise performance  
- Detect loss-making areas and improvement opportunities  

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Dataset
- Source: Superstore Dataset (Kaggle)  
- Features include:
  - Order details (Order ID, Date, Ship Mode)  
  - Customer details (Segment, Region)  
  - Product details (Category, Sub-Category)  
  - Financial metrics (Sales, Profit, Discount)  

---

## Dataset Setup
To run this project:

1. Download the Superstore dataset from dataset folder  
2. Place the dataset file in your project folder  
3. Open `superstore_sales.ipynb` in Jupyter Notebook  
4. Run all cells step by step

## Key Analysis Performed

### Data Cleaning

### Exploratory Data Analysis (EDA)
- Sales and profit distribution

### Visualization
- Sales trends over time  
- Profit distribution  
- Category-wise comparisons  
- Regional performance charts  

---

## Key Insights
- Certain sub-categories generate high sales but low profit  
- Discounts negatively impact profit in many cases  
- Technology category performs best in terms of revenue  
- Some regions contribute more to overall sales than others  
- A few products drive a large portion of total revenue  

---

## Business Recommendations
- Reduce excessive discounting on low-profit products  
- Focus on high-performing categories and regions  
- Optimize inventory for top-selling products  
- Improve strategy for loss-making sub-categories  

---

## Limitations
- Static dataset (no real-time updates)  
- No customer demographic details  
- No predictive modeling applied
